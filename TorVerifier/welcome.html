To enable Tor privacy features, you need to install the helper app.

Step 1: Download the Helper

Download the setup package for your system.

Is this file safe? What does it do?

What is this file?
This is a simple "bridge" application. Browser extensions are blocked from launching external programs (like Tor) for security reasons. This helper app runs locally to start the Tor network on your behalf.

Is it safe?
Yes. The code is open-source, runs entirely on your machine, and does not transmit your data to any third-party servers. It only listens for "Start" and "Stop" commands from this extension.

Step 2: Run the Installer

1. Open the downloaded ZIP file.

2. Double-click install.command inside the folder.

This script automatically sets permissions and launches the proxy.

Help: "File could not be executed / Access Denied"

If macOS says you don't have privileges, follow these steps:

  1. Open your Terminal app (Cmd+Space, type "Terminal").
  2. Type this command (don't hit Enter yet):
    3. chmod +x
  3. Drag and Drop the install.command file from your Finder window into the Terminal window.
  4. Hit Enter.
  5. Now Double-Click the file again. It should work!
Help: "Unidentified Developer" or "Blocked"

If macOS blocks the app because it's from an unidentified developer:

  1. Click OK on the error popup.
  2. Open System Settings and go to Privacy & Security.
  3. Scroll down to the Security section.
  4. You will see a message about "PrivacyLedgerHost" being blocked.
  5. Click the Open Anyway button.
  6. Enter your password if prompted, then click Open.
